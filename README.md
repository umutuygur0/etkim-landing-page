# PDR360 - Landing Page

Modern, güven veren, eğitim odaklı tek sayfa landing page.

## Teknolojiler

- React 18
- Vite
- Tailwind CSS

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Özellikler

- ✅ Sticky Navbar (smooth scroll menü)
- ✅ Hero section (CTA butonları)
- ✅ 2 sütunlu tanıtım bölümü
- ✅ Katkısı Ne? / Neden Bu Ürün? bölümleri
- ✅ Demo videosu bölümü (placeholder)
- ✅ İçerikler bölümü (kartlar)
- ✅ FAQ bölümü (accordion)
- ✅ Ürün Talep Formu (KVKK onayı)
- ✅ Footer (Hakkımızda + İletişim)
- ✅ Responsive tasarım
- ✅ Modern UI/UX

## Bileşenler

- `Navbar.jsx` - Sticky navigasyon menüsü
- `Hero.jsx` - Ana karşılama alanı
- `Tanitim.jsx` - Ürün tanıtım bölümü
- `Katkisi.jsx` - Öğrencilere katkısı ve neden bu ürün
- `Demo.jsx` - Demo videosu bölümü
- `Icerikler.jsx` - İçerik kartları
- `BizKimiz.jsx` - Hakkımızda bölümü
- `FAQ.jsx` - Sıkça sorulan sorular (accordion)
- `TalepFormu.jsx` - Ürün talep formu
- `Footer.jsx` - Footer bölümü
